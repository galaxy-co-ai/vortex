# Vortex Competitive Analysis

## Direct Competitors

### RadarScope — Gold Standard for Pros
- **Strengths:** Level 2/3 NEXRAD, tilt selection, storm-relative velocity, dual-pol, lightning, warning polygons, sub-minute updates
- **Weaknesses:** Utilitarian UI, steep learning curve, zero community features, no streaming, no social, no collaboration
- **Pricing:** $9.99 upfront + subscription for pro features
- **Our advantage:** Free tier with community features they'll never build. RadarScope serves meteorologists; we serve the community around the weather.

### Windy.com — Beautiful but Shallow
- **Strengths:** Gorgeous animated viz, global coverage, 40+ data layers, free, webcam integration
- **Weaknesses:** Radar returns 10-15 dBz low (inaccurate for storm analysis), no Level 2 data, no velocity products, no storm reports, no spotter integration
- **Our advantage:** Accurate radar from IEM/NEXRAD + severe weather focus. Windy is for global weather tourists; we're for tornado country.

### WSV3 — Broadcaster Grade
- **Strengths:** Sub-3.5s delay from real-time, GPU-rendered, 128 GOES-R channels, derived products (hail size, rotation, echo tops), used by TV broadcasters
- **Weaknesses:** Windows-only desktop, no mobile, no web, no community features, complex pro interface
- **Our advantage:** Web-based, free, community-powered. WSV3 is for the TV weather desk; we're for everyone watching at home.

### Storm Shield — Alert-Only
- **Strengths:** Storm-based alerts (not county-based), voice/push notifications, Do Not Disturb override
- **Weaknesses:** Alert-only, minimal radar, no analysis tools, no community
- **Our advantage:** Full radar + analysis + community + alerts. Storm Shield is one feature of our platform.

### MyRadar — Best Casual Experience
- **Strengths:** Hyper-local precipitation, smooth animations, clean UI, social sharing
- **Weaknesses:** No professional radar products, no storm chaser tools, limited historical data
- **Our advantage:** Professional depth with approachable UX. MyRadar tops out where we start.

## Adjacent / Partial Competitors

| Tool | What They Do | Gap We Fill |
|------|-------------|-------------|
| SevereStudios | Live chase streams | No radar, no analysis tools |
| Tornado Archive | 100+ years of historical data | No live weather integration |
| Skywarn/Spotter Network | Ground truth reports, 400K trained spotters | Antiquated tools (HAM radio, phone calls) |
| StormCenter.app | Live chaser map | Limited radar, no community features |
| Storm chaser games (OUTBRK, Storm Chasers) | Gamified experience | Not real weather data |
| SPC.noaa.gov | Authoritative forecasts | Raw text/image products, no consumer UX |

## Our Unique Position

Nobody combines:
1. Professional-grade radar (free, from IEM/NEXRAD)
2. Live community spotting with structured reporting
3. Gamification (achievements, Elo-rated forecasts, leaderboards)
4. Historical tornado paths overlaid on live radar
5. AI storm narratives and cell profiles
6. Virtual chase mode with embedded live streams

Each competitor owns one piece. We own the integration.

## Key Insight

The severe weather community is fragmented across 6+ tools during an active tornado event. A chaser might have RadarScope for radar, SevereStudios for streaming, Twitter for reports, SPC website for outlooks, HAM radio for spotter comms, and a separate GPS app for routing. Consolidating even 3 of those into one platform is a significant win.
